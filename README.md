Project Overview

Implemented and compared multiple face/person detection approaches:

• YOLOv8 Face
• YuNet Face Detector
• MobileNet SSD
• YOLOv8n
• YOLOv8l

Optimization Techniques

• Frame Skipping
• MOSSE Tracking
• CPU Usage Monitoring
• FPS Monitoring
• Memory Monitoring

Results

CPU utilization reduced from 83% to 45%
using hybrid YOLO + MOSSE tracking.
